Arxie

A research operating system for persistent paper work.

Arxie pairs an assistant with a durable paperbase so you can search a field, inspect evidence, compare methods, and return to the same research context later.

Open Workspace

Search a living corpus

Query papers by topic, author, venue, dataset, method, metric, and extraction state instead of repeating shallow API searches every session.

Inspect structured evidence

Open papers as data: sections, figures, tables, methods, datasets, metrics, result rows, and evidence spans tied back to source text.

Keep the research context

Save workspaces on top of curated collections so a search query, pinned papers, and focus note survive beyond a single chat or browser session.
