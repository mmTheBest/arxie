# Arxie User Guide

## Library

Use Library to build and prepare a paper collection.

1. Upload a PDF folder or import papers by identifier.
2. Watch the processing log until import completes.
3. Parse unprocessed papers to make full text searchable.
4. Extract text-ready papers to make structured comparison available.
5. Use paper-level selection when you only want to process specific papers.

## Study

Use Study to work with a prepared collection.

1. Search within the active collection.
2. Inspect papers, chunks, artifacts, figures, tables, and evidence.
3. Save a study so the context can be reused.
4. Add explicit sources such as notes, drafts, code summaries, or result summaries.
5. Ask for experiment ideas, benchmark planning, revision priorities, assumption checks, hypotheses, field patterns, or critiques.

Arxie only uses explicit user-provided study sources. It does not automatically scan your filesystem.

## Readiness Labels

- Imported: paper metadata/file exists.
- Text ready: parsing is complete.
- Evidence ready: structured extraction is complete.
- Needs attention: a preparation job failed or is stale.

Structured comparison works best after extraction is complete.
